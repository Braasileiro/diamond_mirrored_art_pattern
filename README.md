# diamond_mirrored_art_pattern
A fun algorithm that draws a mirrored diamond around bars according to the desired size.

# What Does It Do?

Generates outputs like this:

```
 \        *        /
  \      ***      /
   \    *****    /
    \  *******  /
     \*********/
      \*******/
       \*****/
        \***/
         \*/
         /*\
        /***\
       /*****\
      /*******\
     /*********\
    /  *******  \
   /    *****    \
  /      ***      \
 /        *        \
```
